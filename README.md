## KinopoiskApp
KinopoisApp written in Swift 5.3 using the TMDb API.

## Setup
if you are from Russia, use VPN service to get Json information from TMDb API

## Frameworks and technologies:

- Swift
- UIKit
- CoreData
- MVVM
- Networking
